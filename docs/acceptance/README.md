## 使用规则

本目录只保存已经实际执行的版本验收记录。验收对象始终是一个开发完成、Commit 和构建产物已经冻结的
完整版本，不是某一天、某一周、某个接口或某个开发阶段。必须先完成该版本冻结范围内的全部开发，再复制
下方模板开始正式验收。阶段检查和单项测试不能替代版本验收；没有真实执行结果的项目不能勾选，未通过项
不能写成通过。
