## 7. 已知限制

- 每个应用只绑定一个活动 Chat 模型，每个知识库固定一个 Embedding 模型和版本。
- V1.0 不提供 OCR、扫描 PDF、自动模型路由、完整工单、SLA 或业务写操作。
- `glm-4.7-flash` 默认开启 Thinking；本候选通过模型配置显式关闭，避免输出 Token 全部消耗在
  `reasoning_content` 后得到空正文。服务端也会把空正文标记为失败，不再保存伪成功记录。
- 免费 Provider 曾有一次瞬时上游错误导致 10 次采样提前中断；限流窗口恢复后的正式 10 次复验全部通过。
  当前结果不构成供应商可用性 SLA，生产应配置付费额度、监控和后续版本的备用路由。
- Redis 不保存长期业务事实，备份不包含 Redis；恢复时只清空 `APP_REDIS_URL` 指定的 DB。
- 当前没有 Git Remote，云端 CI 和 `v1.0.0` Tag 尚未执行。
